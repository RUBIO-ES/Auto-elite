# 🚗🛵 Concesionario AutoMoto – Plataforma de Ventas y Cotizaciones

## Descripción
**Concesionario AutoMoto** es una aplicación web moderna que permite a los usuarios explorar, seleccionar y cotizar vehículos de manera fácil e intuitiva. Nuestro portal integra tanto **autos** como **motos**, ofreciendo una experiencia completa para compradores y entusiastas del motor.  

Los usuarios pueden gestionar sus **favoritos**, realizar **cotizaciones personalizadas** y mantenerse al día con nuestras **ofertas exclusivas**, todo en un entorno digital seguro y atractivo.

---

## Características principales

- 🌟 **Catálogo de Vehículos**
  - Autos y motos organizados por marca, modelo y tipo.
  - Búsqueda rápida por características o rango de precios.

- ❤️ **Favoritos**
  - Guarda tus vehículos preferidos para revisarlos más tarde.

- 💰 **Cotizaciones en línea**
  - Calcula precios personalizados según el modelo, versión y accesorios.
  - Genera un resumen que se puede descargar o enviar por correo.

- 🛠 **Gestión de inventario**
  - Administra el stock de vehículos disponibles.
  - Añade, edita o elimina vehículos desde el panel de administración.

- 📱 **Diseño responsivo**
  - Compatible con móviles, tablets y escritorio.
  - Interfaz moderna usando **React** y **Material UI**.

---

## Tecnologías utilizadas

- **Frontend:** React, Material UI, CSS3, HTML5  
- **Estado y contexto:** React Context API (Carrito, Favoritos, Cotizaciones)  
- **Backend (opcional):** Node.js, Express, MongoDB  
- **Herramientas:** Vite, Git
